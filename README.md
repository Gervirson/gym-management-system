# 🏋️ Sistema de Gestión de Usuarios

## 📖 Descripción
Este sistema permite la **gestión controlada de datos de usuarios**, incluyendo:
- Creación, edición, eliminación y visualización de usuarios.
- Cada usuario se guarda en un archivo JSON local con un **ID único**.
- Los usuarios pueden realizar compras, y cada compra se guarda en un archivo separado como recibo, vinculado al ID del usuario.
- Seguimiento de membresías: estado (activo/inactivo), fecha de creación y tipo de membresía.
- Visualización de datos de un usuario o múltiples usuarios, con filtros por diferentes criterios (edad, membresía, estado, fecha de nacimiento).
- Sistema semitraducido a **tres idiomas**: Español, Inglés y Francés.
- Compilado en formato **.exe** y listo para ser usado en entornos de escritorio.

---

## 🎯 Propósito
- Centralizar la información de usuarios.  
- Facilitar consultas rápidas (por ID, membresía, estado, etc.).  
- Practicar conceptos de manejo de archivos JSON en Dart.  
- Servir como base para futuras ampliaciones (reportes, interfaz gráfica, app móvil).  

Este proyecto nació como práctica de programación en **Dart**, con el objetivo de aprender a manejar archivos, estructuras JSON y menús interactivos en consola, aplicados a un caso realista: la gestión de un gimnasio.
